# Тестовое задание - многопоточная реализация записи сообщений в журнал

Разработана Сычуком Алексеем Александровичем

## Сборка

```bash
make        # собрать библиотеку и приложение
make clean  # удалить файлы сборки
```

## Использование

```bash
./logger_app <файл_журнала> <уровень>
```

Уровни: `DEBUG`, `INFO`, `ERROR`.

Пример:

```bash
./logger_app app.log INFO
```

После запуска введите сообщение. Можно указать уровень перед сообщением:

```
> hello
> ERROR server down
```

Введите уровень без сообщения, чтобы изменить уровень по умолчанию и порог:

```
> ERROR
Уровень по умолчанию и пороговый уровень изменены на: ERROR
```

Пустая строка завершает работу.

## Запуск тестов

```bash
make test
```
